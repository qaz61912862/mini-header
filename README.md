# header-navbar
小程序自定义导航栏，左上角胶囊按钮返回|首页详解


详情介绍地址：https://juejin.im/post/5d0219536fb9a07ef06f8eab

# 主要API
**获取设备信息**<br>
`wx.getSystemInfo`

**获得胶囊按钮坐标信息**<br>
`wx.getMenuButtonBoundingClientRect`
